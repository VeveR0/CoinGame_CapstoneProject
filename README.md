# About

Game created as my high school capstone project. Developed only by using the programming knowledge I had learned up to that point.

# Rules

The game consists of two players and a randomly generated row of coins, each having a random value. On each turn, a player can select one of the two coins located at either end of the row. The game ends when no coin remains. The player with the highest score wins.

# Display and Controls

## Main Menu

<img width="890" height="107" alt="image" src="https://github.com/user-attachments/assets/ccc4d8ff-4b26-4b23-bde0-06bcc4d5c861" />

- W / 'UP_ARROW' = Move up
- S / 'DOWN_ARROW' = Move down
- E / 'Enter' = Confirm the selected menu option.

## Game Selection Menu

<img width="310" height="239" alt="image" src="https://github.com/user-attachments/assets/9fb2153a-1d66-46b5-9d20-79f77ca4e652" />

- A / 'LEFT_ARROW'  = Decrease the maximum value and the number of coins that spawn in a game session
- D / 'RIGHT_ARROW' = Increase the maximum value and the number of coins that spawn in a game session
  
## Gameplay

<img width="516" height="83" alt="image" src="https://github.com/user-attachments/assets/a1e488f4-7247-4466-9a4f-ace0522f132f" />

Player 1:

- A – Move the cursor to the left
- D – Move the cursor to the right
- E – Select the desired coin

Player 2:

- 'LEFT_ARROW' = Move the cursor to the left
- 'RIGHT_ARROW' = Move the cursor to the right
- 'Enter' = Select the desired coin
